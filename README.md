# 🐾 Sistema Web Petshop - Fase 1

Este repositório contém o desenvolvimento da primeira etapa do **Projeto da Disciplina**, focado na construção da base estrutural de um sistema web para gerenciamento de um Petshop.

## 📝 Descrição do Projeto
O objetivo desta fase é criar a arquitetura de informação e a navegação do site utilizando exclusivamente a linguagem de marcação **HTML**. 

Conforme os requisitos da **Fase 1**, o projeto prioriza a semântica e a funcionalidade dos elementos de interface, preparando o terreno para a estilização (CSS) e interatividade (JavaScript) que serão implementadas nas fases seguintes.

## 🎯 Objetivos desta Etapa
* **Estruturação Semântica:** Uso de tags HTML5 para organizar cabeçalhos, menus, seções e rodapés.
* **Navegação:** Interligação de todas as páginas do sistema via links.
* **Formulários:** Criação de interfaces para cadastro de pets, clientes e agendamentos.
* **Acessibilidade básica:** Organização hierárquica do conteúdo para leitura de navegadores.

## 🛠️ Tecnologias Utilizadas
* **HTML5** (Sem customizações de CSS ou frameworks como Bootstrap).

## 📂 Estrutura de Arquivos (Sugestão)
* `index.html`: Página principal (Home).
* `cadastro-pet.html`: Formulário de registro de animais.
* `servicos.html`: Listagem de banho, tosa e consultas.
* `contato.html`: Informações de localização e formulário de mensagem.
